#实现功能
> 1. 验证自身Token
> 2. 获取微信access_token
> 3. 获取微信服务器IP
> 4. 接收文本信息自动回复文本信息
> 5. 调用图灵机器人API实现[聊天功能]
> 6. 调用天气API实现[天气功能]
> 7. 调用微信热门API实现[热门功能],回复图文信息
> 8. 接收关注/取消关注事件,保存/删除用户openid
